# Семантические отношения

## Таблицы, хранящие семантические отношения

### Таблица `words`

Хранит список слов, которые знакомы ManSPy.

Поля:
- `word` - слово (поле с уникальными значениями)
- `id_word` - уникальный идентификатор слова
